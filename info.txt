Respuesta: 

Aquí tienes una propuesta completa para una página web sencilla de una cafetería artesanal, usando solo archivos HTML, CSS y JavaScript externos, junto con una explicación de la estructura del proyecto y cómo ejecutarlo en un navegador.

Estructura del Proyecto
El proyecto consta de tres archivos principales:

index.html: El archivo HTML que define la estructura y el contenido de la página web.

style.css: El archivo CSS externo donde se define el diseño, los colores, la fuente y el aspecto visual general.

script.js: El archivo JavaScript externo para añadir interactividad, como cambiar el color de fondo al hacer clic en un botón.

La estructura de carpetas recomendada:

text
cafeteria-artesanal/
│
├── index.html
├── style.css
└── script.js
Paleta de Colores y Fuente
Para evocar el ambiente acogedor de una cafetería artesanal:

Marrón oscuro (#5A3A1B): Marrón café

Beige claro (#F5EFE6): Fondo cálido

Verde olivo (#A3A847): Toques naturales

Blanco puro (#FFFFFF): Para contraste

Fuente: 'Bitter', serif; (disponible desde Google Fonts, ideal para cafetería artesanal)

script.js
javascript
const colorBtn = document.getElementById('colorBtn');
const colores = ["#F5EFE6", "#A3A847", "#FFF", "#E7CBA9", "#FFD9B7"];
let indice = 1;

colorBtn.addEventListener('click', function() {
    document.body.style.backgroundColor = colores[indice];
    indice = (indice + 1) % colores.length;
});
Explicación de la Interacción en JavaScript
Al hacer clic en el botón “Cambiar color de fondo”, el color del fondo de la página irá alternando entre cinco diferentes tonos suaves y cálidos, apropiados para una cafetería artesanal.

Esto se logra seleccionando el botón por su ID (colorBtn), definiendo un arreglo de colores, y cambiando el color de fondo de body cada vez que se hace clic.

Cómo abrir el proyecto en un navegador
Crea una carpeta llamada cafeteria-artesanal.

Guarda cada código provisto arriba en su respectivo archivo dentro de esa carpeta:

index.html

style.css

script.js

Abre el archivo index.html con doble clic o clic derecho y seleccionando "Abrir con" → tu navegador web preferido.

Asegúrate de que los tres archivos estén juntos en la misma carpeta para que se vinculen correctamente.

Esta estructura modular y sencilla permite personalizar la web fácilmente y es ideal para proyectos pequeños o demostraciones.
